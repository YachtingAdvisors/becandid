# Be Candid for Windows — Release Notes

## Version 1.0.0

### What is Be Candid?

Be Candid is a digital accountability application that helps you align your screen time with your values. Unlike surveillance-based tools, Be Candid never logs URLs or captures browsing history. Instead, it provides structured journaling, AI-guided accountability conversations, focus tracking, and partner alerts — all grounded in clinical research.

The desktop app adds full-screen awareness and background heartbeat monitoring to ensure accountability stays active while you work, browse, or study.

### Features

- Background heartbeat monitoring with system tray integration
- Screen awareness powered by AI content analysis
- Conversation Coach for real-time accountability dialogue
- Stringer-based journaling with structured prompts
- Focus Streaks and Digital Fasting schedules
- Cross-platform sync with your mobile and web accounts
- Dark mode support

### Installation Instructions

1. Download the `Be-Candid-Setup-1.0.0.exe` installer.
2. Run the installer. Windows may show a SmartScreen warning (see note below).
3. Follow the one-click NSIS installer — no configuration required.
4. Be Candid will launch automatically after installation.
5. Sign in with your existing Be Candid account, or create a new one at https://becandid.io.

### SmartScreen Warning

This build is not yet signed with an Extended Validation (EV) code signing certificate. When you run the installer for the first time, Windows SmartScreen may display a warning that says "Windows protected your PC."

To proceed:

1. Click **More info** on the SmartScreen dialog.
2. Click **Run anyway**.

This warning will disappear once the application accumulates enough installations to build SmartScreen reputation, or once we obtain an EV code signing certificate. The application is safe to install — the source code is available for review upon request.
